# 📚 Book Management App

A full-stack web application to manage a collection of books with **Create, Read, Update, and Delete (CRUD)** functionality.

---

## 🚀 Tech Stack

**Frontend**
- Next.js (React)
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- PostgreSQL

---

## ✨ Features

- View list of books
- Add a new book
- Edit/update book details
- Delete books
- REST API integration
- Simple UI for managing books
